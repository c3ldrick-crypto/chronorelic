Comment ça fonctionne

Le Système de Runes

Les runes sont des pierres runiques obtenues via des captures spéciales. Seules, elles améliorent légèrement un objet. Combinées dans l'ordre correct dans les socles d'un même équipement, elles activent un Mot Runique — un pouvoir ancien d'une ampleur bien supérieure.

💎
Insertion dans les Socles
Chaque objet avec des socles vides peut recevoir des runes. L'ordre d'insertion est crucial — les mêmes runes dans un ordre différent ne donnent pas le même Mot Runique.
📜
Recettes Secrètes
Les combinaisons de Mots Runiques ne sont pas affichées dans le jeu. Les joueurs les découvrent, les partagent, les gardent secrets. Un méta-jeu de connaissance se crée autour des recettes.
⚖️
Compatibilité par Rareté
Les objets COMMUNE accueillent les Mots Runiques à pleine puissance. Les raretés plus élevées activent des combinaisons mais avec une puissance réduite — les Légendaires préfèrent leurs propres passifs.

Insérez les runes dans l'ordre exact pour activer un Mot Runique. Le pouvoir combiné dépasse largement la somme des runes individuelles. Certaines recettes sont connues de tous — d'autres restent secrètes.

Règles de Compatibilité

Puissance selon la Rareté

Les Mots Runiques fonctionnent dans toutes les raretés, mais leur puissance varie. Un objet COMMUNE ordinaire avec les bonnes runes peut surpasser un Légendaire. C'est le paradoxe des Mots Runiques.

COMMUNE
100%
Pleine puissance · Tous tiers compatibles · La base idéale pour les Mots Runiques
MAGIQUE
85%
Légèrement réduit · Tiers 1–3 uniquement · Bonne alternative intermédiaire
RARE
70%
Réduit · Tiers 1–3 uniquement · Compromis stats propres + mot runique
LÉGENDAIRE
50%
Moitié puissance · Tiers 1–2 seulement · Passif propre reste prioritaire
MYTHIQUE
Runes seules
Runes individuelles uniquement · Pas de Mot Runique · Le Mythique domine toujours
